## Komande:
#### Glavne komande:
`W` - napred <br>
`S` - nazad <br>
`A` - levo <br>
`D` - desno <br>
#### Komande dijamanta:<br>
`P` - pink diamond <br>
`C` - default diamond <br>
`B` - bling diamond <br>
#### Ostale komande:
`F1` - pocetni prozor (hello window) <br>
`F2` - podesavanja
